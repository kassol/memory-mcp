# Memory MCP

Personal memory service with semantic search and knowledge graph, powered by MCP Streamable HTTP.

## Quick Start
1. Copy env template: `cp .env.example .env`
2. Fill in `MEMORY_MCP_AUTH_TOKEN` and OpenRouter settings
3. Start server: `python -m memory_mcp`

## Commands
- Install dependencies: `pip install .`
- Run tests: `pytest`
